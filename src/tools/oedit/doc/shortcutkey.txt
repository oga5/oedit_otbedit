
ショートカットキー

 Esc         : キーワード補完
 Shift + Esc : キーワード補完(逆順)

 Ctrl + A    : 全て選択
 Ctrl + C    : 選択範囲をコピー(Ctrl + Insert)
 Ctrl + X    : 選択範囲をカット(Shift + Delete)
 Ctrl + V    : ペースト(Shift + Insert)
 Ctrl + Z    : Undo
 Ctrl + Y    : Redo

 Ctrl + F    : 文字列検索
 F3          : 次を検索
 Shift + F3  : 前を検索
 Ctrl + H    : 文字列置換
 Alt + F3    : 検索文字列の強調表示を解除
 Ctrl + G    : 指定行へ移動

 Ctrl + N    : 新規ウィンドウを開く
 Ctrl + O    : ファイルを開く
 Ctrl + S    : ファイルに保存
 Ctrl + P    : 印刷
 Alt + F4    : アプリケーションを終了

 Tab         : 選択範囲の各行に，Tabを挿入する
 Shift + Tab : 選択範囲の各行から，Tabを削除する

 Ctrl + U         : 選択中の領域を小文字に変換する
 Ctrl + Shift + U : 選択中の領域を大文字に変換する
 Ctrl + Shift + R : 選択中の各行に，"> "を挿入する
 Ctrl+Shift+Space : 選択中の各行の，先頭の空白を削除
